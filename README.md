# 🗳️ Online Voting System

## 🔧 Tech Stack

* Core Java
* JDBC
* MySQL

## 🚀 Features

* User Registration
* Login Authentication
* One Person One Vote System
* Vote Tracking using Database
* Result Display

## 📂 Project Structure

* model → Data classes
* service → Business logic
* db → Database connection
* main → Entry point

## ▶️ How to Run

1. Setup MySQL database
2. Add MySQL Connector JAR
3. Run MainApp.java
